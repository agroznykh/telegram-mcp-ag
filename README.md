# telegram-mcp-ag

Даёт ассистентам Claude и ChatGPT/Codex читать ваш Telegram и расшифровывать
голосовые сообщения, видео-кружки и аудио. Только чтение: отправлять сообщения
или менять что-либо в вашем аккаунте это не умеет и не будет.

## Что умеет

- Читать чаты, папки и контакты.
- Расшифровывать **голосовые сообщения, видео-кружки и аудиофайлы/подкасты**
  — можно попросить ассистента пересказать их текстом.
- Пачками: можно попросить «перескажи все голосовые за сегодня» — не придётся
  просить по одному сообщению.

**Не умеет:** обычные видео и документы — даже если внутри у них есть звук,
Telegram не даёт их транскрибировать.

## Что подготовить заранее

Для **любого** способа установки ниже понадобятся `api_id` и `api_hash` —
это данные вашего Telegram-приложения, не пароль. Получить их можно бесплатно
на [my.telegram.org/apps](https://my.telegram.org/apps) (нужно один раз войти
по номеру телефона, дальше — заполнить короткую форму создания приложения).
Установщик или ассистент попросят вставить эти два значения в самом начале.

Как получить `api_id` и `api_hash`:

1. Откройте в браузере [my.telegram.org](https://my.telegram.org) и введите
   свой номер телефона в международном формате (например, `+7...`).
2. Telegram пришлёт код входа — не SMS, а сообщение в самом приложении
   Telegram (как код для входа с нового устройства). Введите этот код на сайте.
3. После входа откройте **API development tools** (если это первый вход —
   сайт сам предложит эту форму).
4. Заполните форму создания приложения:
   - **App title** — любое название, видите только вы (например, «Мой ассистент»);
   - **Short name** — короткое имя без пробелов (например, `my-assistant`);
   - **Platform** — выберите любой пункт, например «Desktop» или «Other», на
     работу не влияет;
   - остальные поля необязательны, можно оставить пустыми.
5. Нажмите **Create application**.
6. На открывшейся странице появятся два значения — **App api_id** (число) и
   **App api_hash** (строка из букв и цифр). Их и нужно скопировать и вставить
   туда, где просит установщик или ассистент.

Это данные приложения, а не пароль от аккаунта, но всё равно держите их при
себе — не публикуйте, не отправляйте в чат с ассистентом и не кладите в код.

## Установка

Если раньше не пользовались ни одним из перечисленных ниже — проще и
удобнее всего поставить Claude Cowork (раздел «Claude Cowork (desktop) —
без терминала» ниже) и вести установку только через него, терминал вообще
не понадобится.

### GPT Work (desktop), Claude Code CLI, GPT Codex CLI — через терминал

Один и тот же установщик: сам находит, что из этого у вас уже стоит на
компьютере, и настраивает всё найденное за один запуск (например, если у вас
есть и Claude Code CLI, и GPT Work — не нужно запускать команду дважды).

**macOS и Linux** — вставьте в терминал:

```bash
curl -fsSL https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.sh | bash
```

**Windows** — вставьте в PowerShell:

```powershell
$f = "$env:TEMP\telegram-mcp-ag-install.ps1"; iwr 'https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.ps1' -OutFile $f; & $f
```

Скрипт попросит войти в Telegram — покажет QR-код прямо в терминале.
Откройте Telegram на телефоне, отсканируйте код (Настройки → Устройства →
Подключить устройство) — и всё готово.

После установки перезапустите то приложение, которым пользуетесь (если оно
уже было открыто) — новые инструменты появятся в новом чате. GPT Work
доступен на macOS и Windows (официального Linux-приложения у него нет);
Claude Code CLI и GPT Codex CLI — на всех трёх ОС.

Этот же установщик заодно настраивает и Claude Cowork, если Claude Desktop
уже стоит на компьютере — отдельно его настраивать не нужно, но у него есть
свой, ещё более простой способ ниже (без терминала вообще).

### Claude Cowork (desktop) — без терминала

Работает на macOS, Windows и Linux. Cowork — вкладка в приложении Claude
Desktop; работать с Telegram рекомендуем именно там.

1. Откройте страницу [Releases](https://github.com/agroznykh/telegram-mcp-ag/releases)
   и скачайте файл с расширением `.mcpb`.
2. Откройте скачанный файл двойным кликом — Claude Desktop сам предложит его установить.
3. Заполните два поля (`api_id` и `api_hash` — см. раздел выше) и подтвердите установку.
4. Ассистент попросит войти в Telegram — покажет QR-код прямо в чате.
   Откройте Telegram на телефоне, отсканируйте код (Настройки → Устройства →
   Подключить устройство) — и всё готово. Перезапускать Claude Desktop не
   нужно — инструменты появятся в том же разговоре. Если этого не произошло,
   откройте Settings → Extensions и выключите/включите расширение обратно.
5. **Установите скилл `telegram-digest`** — следующий раздел ниже. Он
   заметно облегчает работу с Telegram в приложении и делает её надёжнее.

Если вы уже настроили Claude Cowork командой в терминале выше — шаги 1-4
не нужны, но шаг 5 (скилл) всё равно выполните — команда в терминале его не
ставит туда автоматически.

### Скилл `telegram-digest` для Claude Cowork (обязательный шаг)

Установите скилл `telegram-digest` — он заметно облегчит работу с Telegram
в приложении. В Cowork его нужно добавить самостоятельно.

**1. Где лежит файл**

- Если ставили командой в терминале выше — файл уже лежит в Загрузках:
  `telegram-digest-skill.zip`.
- Если ставили `.mcpb`-бандл в Claude Desktop — скачайте
  [telegram-digest.zip](https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/.claude/skills/telegram-digest.zip)
  по этой ссылке — он тоже окажется в Загрузках.

**2. Включите скиллы в настройках аккаунта**

В Claude Desktop (или на claude.ai) откройте значок профиля → **Settings**
→ **Capabilities** и включите **«Code execution and file creation»**, если
выключено — без этого скиллы не работают вообще. На рабочем/корпоративном
аккаунте этот переключатель может быть у администратора, в
**Organization settings → Skills**.

**3. Подключите скилл**

Значок профиля → **Settings** → **Customize** → вкладка **Skills** →
**Add** → **Upload a skill** → выберите zip-файл из шага 1. Скилл появится
в списке.

### Что не поддерживается

Веб-версии и мобильные приложения — ни у Claude (claude.ai), ни у ChatGPT —
подключить нельзя, ни на каких ОС.

## Как этим пользоваться

После установки просто спрашивайте у ассистента — Telegram-инструменты
работают как обычная часть разговора, отдельно их вызывать не нужно.
Спросить можно про содержимое **любого** чата, который виден вашему
аккаунту, — личного, группового или канала. Например:

- «Перескажи, что нового в чате „Семья“ за сегодня»
- «Есть голосовые от Артёма за последние два дня? Перескажи текстом»
- «Что писали в рабочем чате, пока меня не было — и включи содержимое кружков»
- «Проверь, сколько у меня осталось бесплатных расшифровок на этой неделе»

Ассистент сам разбирается, какие сообщения — голосовые, кружки или обычный
текст, и при необходимости запускает расшифровку без дополнительных просьб.

## Если ассистент перестал видеть Telegram

Если Telegram-инструменты вдруг перестали работать — скорее всего, сессия
для этого устройства была отозвана (например, вручную в приложении Telegram
через Настройки → Устройства, или разлогинены сразу все сеансы). Сервер сам
распознаёт эту ситуацию и в ответе на любой инструмент сразу называет нужную
команду для этой машины — можно просто спросить ассистента ещё раз, он
покажет её. Та же команда — на случай, если хотите выполнить её сами, без
ассистента (без переустановки клиентов). `api_id`/`api_hash` попросят ввести
ещё раз — те же самые, их можно посмотреть повторно на
[my.telegram.org/apps](https://my.telegram.org/apps):

**macOS и Linux:**

```bash
curl -fsSL https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.sh | bash -s -- --relogin
```

**Windows:**

```powershell
$f = "$env:TEMP\telegram-mcp-ag-install.ps1"; iwr 'https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.ps1' -OutFile $f; & $f -Relogin
```

## Как удалить

**Если ставили через терминал** (Claude Code CLI, GPT Codex CLI, GPT Work, а
также Claude Cowork, если она была настроена тем же установщиком):

macOS и Linux:
```bash
curl -fsSL https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.sh | bash -s -- --uninstall
```

Windows:
```powershell
$f = "$env:TEMP\telegram-mcp-ag-install.ps1"; iwr 'https://raw.githubusercontent.com/agroznykh/telegram-mcp-ag/main/install.ps1' -OutFile $f; & $f -Uninstall
```

**Если ставили Claude Desktop через `.mcpb`-бандл:** откройте Claude Desktop →
Settings → Extensions, найдите telegram-mcp-ag и удалите его там же — так же,
как любое другое расширение.

**В любом случае — отзовите доступ в приложении Telegram:** Настройки → Устройства,
найдите устройство с именем вида `telegram-mcp-ag (<имя компьютера>)` и
завершите его сессию. Это самый быстрый и надёжный способ: сработает
мгновенно, даже если у вас больше нет доступа к компьютеру, где стоял сервер.

## Premium и бесплатная квота на расшифровку

Расшифровка голосовых — это функция самого Telegram, а не этого проекта, и
у неё есть ограничение:

- **С Telegram Premium** — расшифровка без ограничений.
- **Без Premium** — есть небольшая бесплатная квота (обычно около двух
  сообщений в неделю, с ограничением по длительности), которая обновляется
  раз в неделю.

Перед тем как расшифровывать, ассистент сам проверяет остаток квоты — если
она закончилась, вы увидите понятное объяснение и дату, когда квота обновится,
а не непонятную ошибку Telegram.
